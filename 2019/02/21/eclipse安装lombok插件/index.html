eclipse安装lombok插件

2019-02-21
  • 插件

由于一个二次开发项目，使用到 @Slf4j 和 @Data 注解等，需要 eclipse 安装 lombok 插件才能正常编译。所以在这里简要介绍一下 lombok 插件的安装。

环境：windows eclipse

  1. 下载 lombok.jar

  2. 将 lombok.jar 放在eclipse 安装目录下，与 eclipse.ini同级目录。
    (安装目录不能有中文或特殊符号，这个老生常谈了)

  3. 编辑 eclipse.ini 文件，在文件最后加上

  4. 重启 eclipse

  5. clean项目，切记，这样就算是完成了 lombok 的安装

以上就是 lombok 插件最简单的安装过程，但是关于这个插件还是有几句话要说。 一般我是不推荐使用这样的一个插件的。

一般在协同开发或者我这种二次开发的情况下，要求别人去安装一个插件来配合你的编程习惯，这是很不友好的。

二是这个插件大多数人是为了省略持久化类(也就是实体类)中的 get/set 方法，而这个是一个快捷键就能生成的东西，也不在逻辑层中，很少会改动到，并不会有多大影响。

当然，以上只是我个人拙见，喜欢用这个插件的猿不必理会。
